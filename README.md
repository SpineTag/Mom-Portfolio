# Mom's Portfolio — Prototype

This is a small static prototype site to showcase your mom's portfolio.

To view:

1. Open [index.html](index.html) in a browser (double-click or use `Live Server`).

Customization:
- Replace `[Mom's Name]` in `index.html` with her real name.
- Update the bio and project cards in `index.html`.
- Change accent color in `css/styles.css` (`--accent`).

Deploy options: drag the folder to GitHub Pages, Netlify, or Vercel.
